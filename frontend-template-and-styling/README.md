# MAQE Homework Challenge - Template and Styling

#### How to install the project

Use ``` npm ``` or ``` yarn ``` to install package

``` npm install ```

or

``` yarn install ```

#### How to run the project

``` npm run dev ```

or

``` yarn dev ```

You will see the result
Server will start on ``` http://localhost:3000 ```
